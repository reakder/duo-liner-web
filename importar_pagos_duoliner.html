Importar pagos DUO-LINER

Este importador lee el Excel de pagos, busca el pedido correspondiente en MongoDB y crea pagos automáticamente para las filas marcadas como cancelado.

Reglas:
  • Solo importa filas donde status = cancelado.
  • El pago se crea por el valor de TOTAL FACTURA.
  • Relaciona por cliente, total, cantidad y descripción.
  • Evita duplicados si ya existe un pago con el mismo pedido, monto y referencia.
Selecciona el Excel de pagos para analizarlo.
